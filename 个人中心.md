1.兑换彩票 
接口名：/bemp/usercenter/exchangeLottery.action
参数:openId 微信用户ID 
返回结果

```javascript
{
    "result": 1, //兑换结果 1：成功 0：失败
    "data": {
        "gameCode": "201", 
        "voteNums": "14/18/24/25/27/31//02", //彩票号码
        "issue": "2016083" //期号
    },
    "errmsg": "兑换成功"
}
```
